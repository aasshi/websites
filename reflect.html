1. Adjustable Screen Resolution
2. Flexible Layout

I have used table in my website at most because it is the easiest way to represent items in particular width and height. As I had fixed size of image, so I think table is the best option for this type of details. I used one table for my navigation bar which allows menu to be vertically aligned on the side. Moreover, I used table to display my products so on decreasing the width of screen all table contents get compressed as per the width of screen. All of the products on my website is displayed in tables so it becomes easy to code and view as well.
